# Practica 2


# 1. ¿Qué es una Base De Datos Relacional?
Las bases de datos relacionales son un tipo de base de datos que organiza los datos en tablas formadas por filas y columnas 
# 2. ¿Qué es una Tabla en el contexto Base De Datos Relacionales? 
es una estructura que se utiliza para organizar los datos de manera lógica y coherente
# 3. ¿Qué es una relación en el contexto de Base De Datos Relacionales?
Es la conexión que existe entre dos o más tablas mediante una clave común (como una clave primaria y una clave foránea). Permite vincular información relacionada.
# 4. ¿Qué representa una columna en el contexto de Base De Datos Relacionales?
Una columna representa un atributo o característica de los datos dentro de una tabla. Por ejemplo, nombre, edad o correo.
# 5. ¿Qué representa un renglón en el contexto de Base De Datos Relacionales?
Un renglón (o fila) representa un registro completo dentro de una tabla, es decir, un conjunto de datos relacionados a una sola entidad.
# 6. ¿Qué es un motor de base de datos relacional?
Es el software que permite crear, administrar y manipular bases de datos relacionales. Se encarga de almacenar, organizar y recuperar la información.
# 7. ¿Cuáles son los motores de bases de datos relacionales mas populares?
Algunos de los más populares son MySQL, PostgreSQL, Oracle Database y Microsoft SQL Server.
# 8. ¿Cuál es el motor de base datos ligero que no necesita un servidor?
SQLite, porque funciona directamente en el dispositivo sin necesidad de instalar un servidor.
# 9. ¿Qué significa SQL en el contexto de base datos relacionales?
SQL significa “Structured Query Language” (Lenguaje de Consulta Estructurado). Es el lenguaje que se usa para consultar y administrar bases de datos.
# 10. ¿Cuáles son los elementos de un sistema de información basado en base de datos relacionales?
La base de datos (tablas, relaciones).
El motor de base de datos.
El lenguaje SQL.
Las aplicaciones que usan la base de datos.
Los usuarios que interactúan con el sistema.
